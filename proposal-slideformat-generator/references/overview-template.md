# 개요.md Template

Use this template only when the workspace should include an operating summary and file-governance rules.

```md
# 제안서 장표 생성 표준 프로세스

## 1. 목적

본 문서는 RfP 기반 제안서 장표를 일관된 방식으로 생성하기 위한 표준 절차를 정리한 것이다.

## 2. 표준 폴더 구조

- `01.Input_RfP`
- `02.Reference_Templete`
- `03.Reference_Contents_Main`
- `04.Reference_Contents_Assistance`
- `05.Output_Slide`
- `AGENT.md`
- `DESIGN.md`
- `TASK.md`
- `개요.md`

## 3. 전체 작업 흐름

1. 입력자료 정리
2. 기준문서 작성
3. 장표 생성
4. 검수 및 확정

## 4. 단계별 실행 절차

### 4.1 입력자료 정리

- RfP를 `01.Input_RfP`에 저장한다.
- 형식 참고자료를 `02.Reference_Templete`에 저장한다.
- 핵심 참고자료를 `03.Reference_Contents_Main`에 저장한다.
- 보조 참고자료를 `04.Reference_Contents_Assistance`에 저장한다.

### 4.2 기준문서 작성

- `DESIGN.md`: 디자인 및 표현 규칙 정의
- `TASK.md`: 과업 구조와 슬라이드 작성 지시 정의
- `AGENT.md`: 실행 순서와 검수 원칙 정의

## 5. 산출물 관리 방식

- 최신본은 항상 기본 파일명으로 유지한다.
- 이전 버전은 `-oldNN` 규칙으로 보관한다.
- `05.Output_Slide`에서 초안, 수정본, 최종본을 관리한다.
```
